# QUEEN ABIMS - Pairing Site

Generate pairing codes or QR codes for WhatsApp Multi-Device linking.

## Quick Start
cd pair && npm install && npm start

Server runs on http://localhost:8000

## Features
- Pair Code - Enter phone number, get a code
- QR Code - Scan QR to link device
- Auto Session ID generation (Queen-Abims-XXXXXXXXXXXXXXXX)
- Auto-follows QUEEN ABIMS channel on connection

## Deploy to Render
1. New Web Service on render.com
2. Root directory: pair/
3. Build: npm install | Start: npm start

## Mega Upload
Edit mega.js with your Mega.nz credentials for session file uploads.
